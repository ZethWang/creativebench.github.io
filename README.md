# CreativeBench Homepage

This repository contains the source code for the CreativeBench project website.

- Homepage source: `public/index_home.html`
- Leaderboard source: `public/index.html` and `src/index.tsx`
- Build output: `build/`

The current version updates the copied template to CreativeBench and leaves the benchmark content as placeholders until the final materials are ready.
